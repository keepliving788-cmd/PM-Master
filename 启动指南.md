## 🎨 配置优化

### 高精度模式（当前配置）

`config/config.yaml` 已配置为高精度模式：

```yaml
indexing:
  embedding:
    model_name: moka-ai/m3e-base  # 中文优化，768维

retrieval:
  weights:
    vector: 0.6      # 语义权重
    keyword: 0.4     # 关键词权重
  rerank:
    enabled: true    # 启用重排序
```

### 如果需要更快速度

编辑 `config/config.yaml`:

```yaml
retrieval:
  rerank:
    enabled: false  # 禁用重排序，延迟从350ms降到120ms
```

## 🐛 常见问题

### Q1: 依赖安装失败？

```bash
# 使用镜像源
pip3 install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q2: 飞书连接失败？

检查：
1. APP_ID 和 SECRET 是否正确
2. 应用权限是否已开通（`wiki:wiki:readonly`, `docx:document:readonly`）
3. 应用是否已发布
4. 网络连接是否正常

### Q3: 索引构建卡住？

```bash
# 查看日志
tail -f logs/feishu-pkb.log

# 强制重建
rm -rf data/chromadb
python3 src/main.py index --full --force
```

### Q4: 检索没有结果？

1. 确认索引已构建：
   ```bash
   ls data/chromadb/
   python3 src/main.py stats
   ```

2. 尝试不同检索模式：
   ```bash
   python3 src/main.py search "查询" --mode=keyword
   python3 src/main.py search "查询" --mode=vector
   python3 src/main.py search "查询" --mode=hybrid
   ```
