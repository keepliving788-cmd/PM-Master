## 🚀 使用方法

### 基本使用

```bash
# 在 Claude Code 中
/feishu-kb search 马来西亚Beez

# 或直接运行
python3 skill_main.py search "马来西亚Beez"
```

### 高级选项

```bash
# 禁用图片检测（更快）
python3 skill_main.py search "查询内容" --no-images

# 返回更多结果
python3 skill_main.py search "查询内容" --top-k 10

# 选择检索模式
python3 skill_main.py search "查询内容" --mode hybrid

# 查看状态
python3 skill_main.py status
```

## ⚙️ 技术架构

### 核心模块

```
feishu-kb/
├── skill_main.py                       # Skill 入口
├── src/
│   ├── utils/
│   │   ├── config.py                   # 配置管理
│   │   ├── smart_chunker.py            # 智能分块
│   │   └── image_handler.py            # 图片处理 ⭐新增
│   └── retriever/
│       └── simple_search.py            # 检索引擎 ⭐增强
├── data/
│   ├── vectors.npz                     # TF-IDF向量
│   ├── bm25_index.pkl                  # BM25索引
│   ├── kb_data.db                      # SQLite数据库
│   └── raw/                            # 原始文档
└── docs/                               # 文档 ⭐丰富
```

### 数据流

```
用户查询
    ↓
[文本检索] TF-IDF + BM25 → RRF融合
    ↓
[图片检测] 正则匹配图片引用 ⭐新增
    ├─ 有图片 → 标注信息
    └─ 无图片 → 保持原样
    ↓
(可选) [图片理解] Claude API ⭐待配置
    ├─ 下载图片
    ├─ 理解内容
    └─ 返回描述
    ↓
返回完整结果
```
