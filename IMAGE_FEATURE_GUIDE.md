## ⚙️ 配置说明

### 当前功能（无需配置）

**图片检测** 默认开启，无需任何配置：
- ✅ 自动识别图片引用
- ✅ 统计图片数量
- ✅ 标注图片位置

### 进阶功能（需要配置）

**图片理解** 需要配置 Claude API：

#### 1. 获取 Anthropic API Key

访问: https://console.anthropic.com/
- 注册/登录账号
- 创建 API Key
- 复制 Key

#### 2. 配置环境变量

编辑 `.env` 文件：
```bash
# 飞书配置（已有）
FEISHU_APP_ID=your_app_id
FEISHU_APP_SECRET=your_app_secret
FEISHU_WIKI_SPACE_ID=your_space_id

# Claude API 配置（新增）
ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxxxxxxx
```

或者使用环境变量：
```bash
export ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxxxxxxx
```

#### 3. 安装依赖

```bash
pip install anthropic>=0.40.0
```

#### 4. 启用完整功能

配置完成后，将自动启用图片理解功能：
- 自动下载图片
- 调用 Claude API 理解内容
- 返回图片描述

## 💰 成本说明

### 当前版本（图片检测）

**成本**: **$0**
- 仅检测图片引用（文本处理）
- 不调用任何 API
- 完全免费

### 未来版本（图片理解）

**成本**: 按实际使用付费

#### 计费方式

```
单次图片理解成本:
- 输入: ~200 tokens ($0.003/1K) = $0.0006
- 输出: ~150 tokens ($0.015/1K) = $0.00225
- 总计: ~$0.003/图片
```

#### 实际成本预估

| 使用场景 | 图片数 | 成本 |
|---------|--------|------|
| 单次查询（包含1张图）| 1 | $0.003 |
| 单次查询（包含3张图）| 3 | $0.009 |
| 100次查询（平均2张图/次）| 200 | $0.60 |

**智能缓存**（自动）:

- 后续查询使用缓存
- 大幅降低重复成本

## 🔧 技术实现

### 架构图

```
用户查询
    ↓
[1] 文本检索 (TF-IDF + BM25)
    ↓
[2] 图片检测
    ├─ 检测图片引用
    ├─ 提取图片文件名
    └─ 统计图片数量
    ↓
[3] 标注输出
    └─ "⚠️ 此段落包含 N 张图片"

（未来扩展）
[4] 图片理解（可选）
    ├─ 通过飞书 MCP 获取图片
    ├─ 调用 Claude API 理解
    └─ 返回图片描述
```

### 代码结构

```
src/
├── utils/
│   └── image_handler.py          # 图片处理模块
│       ├── ImageHandler          # 飞书图片获取
│       └── ClaudeImageUnderstanding  # Claude 理解器
│
└── retriever/
    └── simple_search.py           # 检索引擎
        └── enrich_with_images()   # 图片增强功能
```

## 🐛 故障排查

### 问题 1: 图片检测不工作

**症状**: 明明有图片，但没有看到提示

**检查**:
```bash
# 1. 确认图片检测已开启
python3 skill_main.py search "查询" --with-images

# 2. 检查日志
# 应该看到: "检测到 N 个图片引用"
```

### 问题 2: 未来图片理解功能无法使用

**症状**: 提示 "Claude 客户端未配置"

**解决**:
```bash
# 1. 检查 API Key
echo $ANTHROPIC_API_KEY

# 2. 或检查 .env 文件
cat .env | grep ANTHROPIC

# 3. 如果没有，添加配置
echo "ANTHROPIC_API_KEY=sk-ant-api03-xxxx" >> .env
```

### 问题 3: anthropic 模块未安装

**症状**: `ModuleNotFoundError: No module named 'anthropic'`

**解决**:
```bash
pip install anthropic>=0.40.0
```
